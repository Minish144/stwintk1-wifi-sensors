Purpose

The STSW-STWINKT1 firmware package for STWIN development kit (STEVAL-STWINKT1) provides sample projects for the development of custom applications. Built on STM32Cube software technology, it includes all the low level drivers to manage the on-board devices and system-level interfaces. The package comes with the Serial_DataLog, WiFi_Connectivity and BLE_SampleApp applications.

  • The Serial_DataLog application shows how to stream sensor data via USB Virtual COM Port class. Data can be showed on a PC Terminal.
  • The BLE_SampleApp provides an example of Bluetooth Low Energy configuration that enables STWIN to stream environmental sensor data; it is compatible with the STBLESensor app available for Android and iOS.
  • The WiFi_Connectivity example implements basic network function using the connectivity framework. This example requires STEVAL-STWINWFV1 Wi-Fi expansion.

This firmware package includes Components Device Drivers, Board Support Package and example applications for the STWIN development kit (STEVAL-STWINKT1).
